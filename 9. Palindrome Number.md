# 9. Palindrome Number
#### Approach 1. covert to string
分析：
Time complexity: O(N), N is length of string
Space cmplexity: O(1)
```c++
class Solution {
public:
    bool isPalindrome(int x) {
        stringstream ss;
        ss << x;
        string str;
        ss >> str;
        for(int i=0 ; i<str.length()/2 ; i++){
            if(str[i] != str[str.length()-1-i]) return false;
        }
        return true;
    }
};
```

#### Approach 2. only use integer
先算出整數長度，每次比完頭尾就把頭尾都去掉
- / 長度^10可以取得頭
- / % 10可以取得尾
- 去掉頭尾長度會直接少了10^2

分析：
Time complexity: O(log_10(N))，N是integer
Space cmplexity: O(1)

```c++
class Solution {
public:
    bool isPalindrome(int x) {
        int temp = x;
        int len=1;
        if(x < 0) return false;
        while(temp /= 10){
            len*=10;
        }
        while(x){
            int left = x / len;
            int right = x % 10;
            if(left != right) return false;
            x %= len;
            x /= 10;
            len /= 100;
        }
        return true;
    }
};
```
